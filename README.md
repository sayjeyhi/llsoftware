## What is this repository for?
This is the Long Lasting Software website. It is built with React and Three.js.
You can find the website at [llsoftware.co](https://llsoftware.co).

## How do I get set up?
### Prerequisites
- NodeJs
- Yarn
- Git
- A text editor
- A terminal
- A browser

### Installation
1. Clone the repository
2. Run `yarn install` to install the dependencies
3. Run `yarn start` to start the development server
4. Open your browser and go to `http://localhost:3000`
